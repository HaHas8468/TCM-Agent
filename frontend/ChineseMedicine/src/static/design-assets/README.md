# Design Assets Pack

This folder contains a first curated batch of reusable design assets for the current project.

## Folder Structure

- `icons/lucide/`
  - UI icons for app screens and feature cards
  - Suggested use: tab bar, cards, status, quick actions
- `illustrations/undraw/`
  - Friendly medical / wellness illustrations
  - Suggested use: onboarding, empty states, service intro blocks
- `reference-photos/wikimedia/`
  - TCM-related reference photos for moodboards and visual direction
  - Suggested use: visual reference, not necessarily direct in-app use

## Recommended Usage For This Project

For the current TCM consultation product, a practical starting combination is:

- use Lucide icons in the real UI
- use unDraw illustrations in onboarding / empty states / service sections
- use Wikimedia photos only as reference for visual tone, texture, props, and atmosphere
